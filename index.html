About Slice 4D Shooter

Slice 4D Shooter is the first multiplayer competitive first person shooter set in true 4D space. Learn how to play a shooter in 4D space in the video tutorial below.

Slice 4D Shooter is being developed by a small indie team of developers since 2023 year on a custom-built 4D game engine with the Rust programming language. All source code of the game is open under a free modified GPL license .

Currently, Slice 4D Shooter is in the playable prototype stage of development and you can try it on itch.io . The current version supports online multiplayer.

We want to make Slice 4D Shooter the first multiplayer competitive esport online shooter set in four-dimensional space, supported by community funding. You can find the development road map here .
